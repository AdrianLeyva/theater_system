# MyTheatherApp

 **C&E Project**  
- 

**MyTheatherApp** manages all sale and buy processes like:

* Ticket office
* Register shows
* Updates and cancellation process
* Storage of each sale
  
  
 
**Architecture:**  
- Model - View - Controller (**MVC**)

**Persistence**  

- Postgresql
